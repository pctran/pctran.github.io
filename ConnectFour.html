Connect Four

Summary:

Developed in Java, I created this program which plays a game of Connect 4. It is played in console and takes input for column number.

Full Download

Start-up:


On start-up, you decide if you want to make the starting move.


AI:


The AI responds by placing a piece in a random column after you make your move.


Game end:


The game ends when four of the same kind of piece are placed in a row horizontally, vertically, or diagnonally.
